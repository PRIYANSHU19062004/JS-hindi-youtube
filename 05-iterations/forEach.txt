What is forEach()?
forEach() is a loop method in JavaScript used to iterate over the elements of an array.
It runs a function once for each element in the array.
Objects can't be directly iterated by for each loop.
It does not return anything â€” itâ€™s mainly for side effects (like printing or performing actions).
You cannot use break or continue inside forEach().
If you need to stop looping early, use a normal for or for...of loop instead.
*****************************//**************************************************
Simple Array
const fruits = ["apple", "banana", "mango"];
fruits.forEach(function(item) {
  console.log(item);
});

Output:

apple
banana
mango

Here, forEach() runs once for each item in the fruits array.
